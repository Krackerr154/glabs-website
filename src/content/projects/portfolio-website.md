## Overview

This personal portfolio website showcases my work, research, and technical writing. Built with performance and accessibility in mind, it leverages modern web technologies to deliver a fast, user-friendly experience.

## Key Features

- **Static Site Generation**: Lightning-fast load times with Astro's static site generation
- **Content Collections**: Markdown-based content management for projects and blog posts
- **Responsive Design**: Mobile-first approach ensuring great experience across all devices
- **Accessibility**: WCAG-AA compliant with proper semantic HTML and ARIA labels
- **SEO Optimized**: Open Graph tags, JSON-LD structured data, and optimized meta tags
- **Contact Integration**: n8n webhook integration for contact form submissions

## Technical Highlights

### Performance Optimization

The site is optimized for fast load times with minimal JavaScript:
- Lazy loading for images
- Optimized asset delivery
- Minimal client-side JavaScript
- Fast LCP (Largest Contentful Paint)

### Content Management

Using Astro's content collections provides type-safe content management with:
- Frontmatter validation
- Automatic routing
- Dynamic page generation

### Deployment

Containerized with Docker for easy deployment and consistent environments across development and production.

## Lessons Learned

Building this project reinforced the importance of:
- Starting with performance in mind
- Progressive enhancement
- Accessibility as a core feature, not an afterthought
- The power of static site generation for content-focused sites
